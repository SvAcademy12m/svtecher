# SVTECH Digital Technology (svtecher)

Welcome to the SVTECH Digital Technology platform. This repository contains the source code for our modern, high-performance digital marketplace and training hub.

## 🚀 Features

- **Multi-Role Experience**: Custom dashboards for Students, Job Finders, Software Buyers, and Administrators.
- **Modern UI**: High-impact design with Blue 700 gradients and a sleek white theme.
- **Course Marketplace**: Premium technology training modules.
- **Job Board**: Direct connections for tech talent and recruiters.
- **Real-Time Data**: Live statistics and analytics powered by Firebase.
- **Secure Payments**: Integrated features for asset acquisition and service enrollment.

## 🛠️ Tech Stack

- **Frontend**: React.js + Vite
- **Styling**: Tailwind CSS
- **Backend/DB**: Firebase Firestore & Authentication
- **Animations**: Framer Motion
